# Моделирование BER для QAM в канале AWGN

## Краткое описание

Репозиторий содержит два компонента:

- **tasks** — модуляторы и демодуляторы QPSK, 16-QAM и 64-QAM, аддитивный белый гауссовский шум (AWGN) и вычисление вероятности битовых ошибок (BER) для значений дисперсии шума (Variance) в диапазоне [0.1; 0.5].
- **chart_plotter** — скрипт на python, который читает JSON-файл с результатами и строит график зависимости вероятности битовых ошибок от дисперсии шума.

После запуска `main.cpp` программы создаётся файл `ber_results.json`, который затем используется для построения графика скриптом `plot_chart.py`.

## Требования

### tasks, C++

- CMake 3.8 или выше
- Компилятор с поддержкой C++20

### chart_plotter, Python

- Python 3.8+
- Модуль matplotlib

Установка зависимости:

```bash
pip install matplotlib
```

## Работа с проектом

### 1. Соберите проект

Из корня репозитория выполните:

```bash
cmake -B build
cmake --build build
```

### 2. Запустите исполняемый файл

После сборки запускается бинарный файл `tasks`.

Пример пути для Linux/macOS:

```bash
./build/tasks/tasks
```

Пример пути для Windows:

```bash
build\tasks\Debug\tasks.exe
```

### 3. Результат работы

После выполнения программы в корне проекта появится файл:

```bash
ber_results.json
```

В нём хранятся рассчитанные значения BER для разных режимов модуляции и значений дисперсии шума.

### 4. Построение графика

Для сгенерированного `ber_results.json` запустите скрипт `charts_plotter/plot_chart.py` и передайте на вход путь до файла через аргумент `json`:

```bash
python charts_plotter/plot_chart.py --json ber_results.json
```

После запуска откроется окно с графиком зависимости `BER` от `Variance` для:

- QPSK
- 16-QAM
- 64-QAM

## Что делает проект

1. Генерирует случайные биты заданной длины.
2. Модулирует их в форматы QAM: QPSK, 16QAM, 64QAM.
3. Пропускает сигнал через канал AWGN с разным значением дисперсии шума.
4. Демодулирует сигнал.
5. Считает вероятности битовых ошибок (BER) и сохраняет результаты в JSON.
6. Строит график зависимости BER от дисперсии шума по полученным данным.
